# Changelog

## [1.0.0] - 2024-01-01

### Added
- `scan` command for scanning common and custom port ranges
- Well-known service name labelling (SSH, HTTP, MySQL, Redis, etc.)
- Concurrent port scanning with configurable batch size
- Progress indicator for large scans
- `report` command with JSON output support
- GitHub Actions CI/CD workflow
- Codespace devcontainer configuration
- GitHub achievement unlock scripts
